## Entry Points for Contributors

**Start anywhere. Fork freely. No permission needed.**

### Story Entry Points

- Write a scene where a character discovers a hidden, thriving ecosystem in an abandoned urban infrastructure (e.g., a subway vent, a closed factory).
- Draft a community meeting where residents with opposing views on 'rewilding' their block clash.
- Create a journal entry from the perspective of a city planner secretly inspired by Leo and Maya's project.
- Write a vignette about a non-human resident (the cat, a rat, a pioneering plant) of 447 Morgan Street, observing the changes.

### Technical Entry Points

- Design a distributed, low-cost stormwater management system for a specific, real urban neighborhood, using the novel's principles.
- Create a 'zine or guidebook titled 'The Guerrilla Gardener's Handbook to Urban Hydrology', with actionable steps and illustrations.
